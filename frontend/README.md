# Vitrine PDV Frontend

Aplicativo mobile criado com [Expo](https://expo.dev) e [Expo Router](https://docs.expo.dev/router/introduction).

## Como rodar

1. Instale as dependências:

   ```bash
   npm install
   ```

2. Inicie o servidor de desenvolvimento:

   ```bash
   npx expo start
   ```

3. Abra no emulador, dispositivo físico com Expo Go ou navegador.

## Desenvolvimento

Comece editando os arquivos em `app/`.
